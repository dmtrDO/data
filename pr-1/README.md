# Опис
Цей код завантажує HTML-сторінки за списком URL-адрес,
парсить їх за допомогою BeautifulSoup,
витягує текст зі сторінки,
очищає його від зайвих пробілів та HTML-елементів,
видаляє слова які не мають змістовної частини (українські та англійські),
визначає топ-20 найчастіших слів для кожного сайту.

# Використані бібліотеки
+ requests — для HTTP-запитів
+ BeautifulSoup (bs4) — для парсингу HTML
+ re — для обробки тексту регулярними виразами
+ collections.Counter — для підрахунку частоти слів
# Нащо це потрібно
Це корисно для індексації пошукових систем, більшість яких працюють наступним чином: спочатку Crawler (павук), який ходить по сайтам, збирає усі посилання і додає їх до бази даних. Потім ця база індексується, тобто витягується текст з усіх посилань, фільтрується, аналізується і отримана структура зберігається в окремій бд. І нарешті ранжування, коли користувач вводить пошуковий запит і програма оцінює за певними факторами, які посилання показати першими.
