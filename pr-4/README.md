# Байєсівський аналіз споживання

Задача полягає у тому, щоб порівняти результати двох моделей: лінійної регресії та баєсівської регресії на даних по енергоспоживанню. <br>

Був обраний наступний датасет
<br>
 https://www.kaggle.com/datasets/sudhirsingh27/electricity-consumption-based-on-weather-data
<br>

тут є наступні стовпці:
<br>date - дата
<br>AWND - середня швидкість вітру
<br>PRCP - вологість
<br>TMAX - максимальна температура
<br>TMIN - мінімальна температура
<br>daily_consumption - цільова змінна - споживання енергії за день
<br>

Спочатку на потрібно підготувати дані, для цього ми очищаємо дані, перемішуємо їх і ділимо на тренувальну та тестову вибірки. Далі для навчання моделей, потрібно стандартизувати або нормалізувати дані, щоб процес навчання проходив якнайкраще.
<br>
Нормалізація - перетворення значень у фіксований діапазон, зазвичай від 0 до 1. Але цей спосіб чутливий до аномалних значень.
<br>
Стандартизація - середнє значення 0, відхилення - 1. Більш стійка, зберігаючи масштаб основних даних.
<br>
Після цього був створений клас для лінійної та баєсівської регресій.
<br>
Лінійна регресія шукає одну ідеальну лінію та видає конкретне число як прогноз.
<br>
Байєсівська регресія шукає розподіл імовірних ліній. Вона видає прогноз разом із відхиленням, показуючи, наскільки можна довіряти результату в кожній точці.
